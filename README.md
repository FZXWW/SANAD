# SANAD Invest AI Web

واجهة ويب عربية لمشروع SANAD Invest AI. التطبيق يستخدم ملف `SANAD_AI_FULL_SYSTEM.py`
والداتاسيت الموجودة لتدريب النماذج ثم تقديم توصية استثمارية للمستخدم.

## التشغيل

```powershell
pip install -r requirements.txt
python app.py
```

ثم افتح:

```text
http://127.0.0.1:5000
```

## آلية العمل

- عند تشغيل السيرفر يبدأ تدريب نماذج القرار والأسهم والصناديق.
- الواجهة ترسل بيانات المستخدم إلى `POST /api/recommend`.
- الخادم يرجع المجال الأفضل، ثقة النموذج، وقائمة التوصيات النهائية.

> هذا مشروع أكاديمي/تجريبي، وليس نصيحة مالية ملزمة.

## النشر العام

تمت إضافة ملفات النشر:

- `Procfile`
- `render.yaml`
- `Dockerfile`
- `DEPLOYMENT.md`

راجع `DEPLOYMENT.md` لخطوات رفع الموقع وربط الدومين.
